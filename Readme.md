# Customer Churn Prediction

## 📌 Project Overview

This project aims to predict whether a customer will churn (leave the service) or not using Machine Learning techniques.
Customer churn prediction helps companies identify customers who are likely to stop using their services and take preventive actions.

---

## 🎯 Objective

* Analyze customer data
* Build a machine learning model
* Predict customer churn (Yes/No)

---

## 📂 Dataset

* Source: Kaggle (Telco Customer Churn Dataset)
* Total Records: 7043
* Features: 21

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

## 🔍 Data Preprocessing

* Removed unnecessary column (`customerID`)
* Converted `TotalCharges` to numeric
* Handled missing values
* Encoded categorical variables
* Converted target variable (`Churn`) into numerical format

---

## 📊 Data Visualization

* Churn distribution analysis
* Used count plots and graphs

---

## 🤖 Machine Learning Model

* Algorithm: Logistic Regression
* Train-Test Split: 80% training, 20% testing

---

## 📈 Model Performance

* Accuracy:78.75 %

---

## 🧠 Conclusion

The model successfully predicts customer churn with good accuracy.
This can help businesses improve customer retention by identifying potential churn customers in advance.
